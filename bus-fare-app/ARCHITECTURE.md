## рҹ§® Distance Calculation (Haversine)

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ                   HAVERSINE FORMULA                              в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

Input:
  Point 1: (lat1, lng1) - Entry Location
  Point 2: (lat2, lng2) - Exit Location

Step 1: Convert to Radians
  lat1Rad = lat1 Г— (ПҖ / 180)
  lat2Rad = lat2 Г— (ПҖ / 180)
  О”latRad = (lat2 - lat1) Г— (ПҖ / 180)
  О”lngRad = (lng2 - lng1) Г— (ПҖ / 180)

Step 2: Calculate 'a'
  a = sinВІ(О”latRad/2) + cos(lat1Rad) Г— cos(lat2Rad) Г— sinВІ(О”lngRad/2)

Step 3: Calculate 'c'
  c = 2 Г— atan2(вҲҡa, вҲҡ(1-a))

Step 4: Calculate Distance
  distance = R Г— c
  where R = 6371 km (Earth's radius)

Output:
  distance in kilometers (rounded to 2 decimals)

Example:
  Entry:  (8.0883, 77.5385)  - Nagercoil
  Exit:   (13.0827, 80.2707) - Chennai
  
  Distance вүҲ 612.25 km
  Fare = 612.25 Г— 2 = вӮ№1,224.50
```
